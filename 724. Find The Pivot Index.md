# Intuition

A pivot index is an index where the sum of all elements on the left is equal to the sum of all elements on the right.

Instead of calculating the left and right sums separately for every index (which would be slow), we first build a **prefix sum array**. The prefix sum array stores the cumulative sum of elements from the beginning of the array.

Using the prefix sum array:

* The left sum of index `i` is `prefix[i]`.
* The right sum of index `i` is `prefix[n] - prefix[i + 1]`, where `prefix[n]` is the total sum of the array.

This allows us to check every index in constant time.

# Approach

1. Create a prefix sum array of size `n + 1` initialized with `0`.
2. Build the prefix sum array using:

   * `prefix[i + 1] = prefix[i] + nums[i]`
3. Traverse every index `i`:

   * Compute the left sum as `prefix[i]`.
   * Compute the right sum as `prefix[n] - prefix[i + 1]`.
4. If the left and right sums are equal, return the current index.
5. If no pivot index exists, return `-1`.

# Complexity

* **Time complexity:** `O(n)`

  * Building the prefix sum takes `O(n)`.
  * Traversing the array once also takes `O(n)`.
  * Overall complexity is `O(n)`.

* **Space complexity:** `O(n)`

  * An additional prefix sum array of size `n + 1` is used.

# Code

```cpp
class Solution {
public:
    int pivotIndex(vector<int>& nums) {

        vector<int> prefix(nums.size() + 1, 0);

        for (int i = 0; i < nums.size(); i++) {
            prefix[i + 1] = prefix[i] + nums[i];
        }

        for (int i = 0; i < nums.size(); i++) {

            int left = prefix[i];
            int right = prefix[nums.size()] - prefix[i + 1];

            if (left == right) {
                return i;
            }
        }

        return -1;
    }
};
```
